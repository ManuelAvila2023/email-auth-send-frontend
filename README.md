# Email Auth Frontend

A React-based authentication frontend with complete user management features including email verification and password recovery.

## Features

- **User Authentication**: Login and registration with JWT tokens
- **Email Verification**: Account activation via email confirmation
- **Password Recovery**: Reset password through email link
- **Protected Routes**: Route guards for authenticated content
- **State Management**: Redux Toolkit for global state
- **Responsive UI**: React Bootstrap with Bootswatch theming

## Tech Stack

- React 18
- Vite
- Redux Toolkit
- React Router v6
- React Hook Form
- React Bootstrap
- Axios

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Environment

Configure the API base URL in `src/utils/axios.js` to point to your backend server.
